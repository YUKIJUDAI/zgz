## 测试场景 8: 缓存过期（7天）

**步骤**:
1. 创建一个过期的缓存条目：
   ```javascript
   const scores = await new Promise(r => chrome.storage.local.get(['bossJobScores'], r));
   scores.bossJobScores['job_expired'] = {
     jobId: 'expired',
     preliminaryScore: { score: 20 },
     lastUpdated: Date.now() - 8 * 24 * 60 * 60 * 1000  // 8天前
   ```
