# Boss直聘求职助手 - 配置说明文档

## 📋 配置文件介绍

`config.js` 是扩展的核心配置文件，包含了所有关键参数。通过修改这个文件，您可以：

- ⚡ 快速调整匹配规则和评分权重
- 🛡️ 自定义防检测行为参数
- 🎨 调整UI显示效果
- 🔧 配置API密钥和服务

**修改后需要**：重新加载扩展（`chrome://extensions/` → 刷新按钮）

---

## 🎯 快速配置指南

### 1️⃣ 修改个人默认配置

**位置**: `BossConfig.defaults`

```javascript
defaults: {
  // 基本信息
  salaryMin: 25000,  // 👈 改成你的期望薪资

  requiredSkills: [
    'Vue', 'TypeScript', 'JavaScript'  // 👈 改成你的核心技能
  ],

  excludedKeywords: [
    '外包', '驻场', '996'  // 👈 添加你想排除的词
  ],

  dailyLimit: 30,  // 👈 每日打招呼上限

  // 个人资料
  profile: {
    name: '金超宇',  // 👈 改成你的名字
    yearsExperience: 9,  // 👈 工作年限
    techStack: 'Vue3/TypeScript/JavaScript',  // 👈 技术栈
    phone: '18557519506',  // 👈 电话
    expectedSalary: '25-35',  // 👈 期望薪资范围
  },
}
```

### 2️⃣ 调整匹配评分权重

**位置**: `BossConfig.scoring`

```javascript
scoring: {
  maxScores: {
    skill: 50,      // 👈 必备技能最高50分
    bonus: 20,      // 👈 加分技能最高20分
    salary: 15,     // 👈 薪资匹配最高15分
    location: 10,   // 👈 地点匹配最高10分
    title: 5,       // 👈 标题匹配最高5分
  },

  bonusSkillScore: 5,  // 👈 每个加分技能给5分

  levels: {
    excellent: 80,  // 👈 80分以上为优秀（绿色高亮）
    good: 60,       // 👈 60-79分为良好（蓝色标签）
  },
}
```

**示例**：如果你更看重薪资，可以这样调整：

```javascript
maxScores: {
  skill: 40,      // 降低技能权重
  bonus: 15,      // 降低加分技能权重
  salary: 30,     // 提高薪资权重 ⬆️
  location: 10,
  title: 5,
}
```

### 3️⃣ 调整防检测参数

**位置**: `BossConfig.behavior`

```javascript
behavior: {
  delays: {
    pageLoad: { min: 1000, max: 2000 },    // 👈 页面加载后等待1-2秒
    beforeClick: { min: 800, max: 2000 },  // 👈 点击前等待0.8-2秒
    typing: { min: 30, max: 120 },         // 👈 打字速度（每个字符30-120毫秒）
  },

  limits: {
    dailyGreet: 30,        // 👈 硬限制：每天最多30次
    maxContinuous: 5,      // 👈 连续操作5次后休息
  },
}
```

**建议**：
- 如果觉得操作太慢，可以减小延迟的 `max` 值
- 如果担心被检测，可以增大延迟的 `min` 值
- 不建议将 `dailyGreet` 设置超过50

### 4️⃣ 配置API密钥

#### OpenAI API

**位置**: `BossConfig.ai.openai`

```javascript
ai: {
  enabled: true,  // 👈 启用AI功能
  provider: 'openai',

  openai: {
    apiKey: 'sk-xxxxxxxxxxxxxxx',  // 👈 填入你的API Key
    baseURL: 'https://api.openai.com/v1',
    model: 'gpt-4o-mini',  // 可选：gpt-4, gpt-3.5-turbo等
  },
}
```

#### 高德地图API

**位置**: `BossConfig.distance.gaode`

```javascript
distance: {
  gaode: {
    apiKey: 'your-key-here',  // 👈 填入高德API Key
    enabled: true,  // 👈 启用距离评估
  },

  homeAddress: '杭州市拱墅区都市水乡水曲苑',  // 👈 你的家庭地址

  limits: {
    maxCommuteTime: 60,    // 👈 最大通勤时间60分钟
    maxDistance: 20000,    // 👈 最大直线距离20km
  },

  preferredAreas: ['西湖', '拱墅', '余杭'],  // 👈 期望工作区域
}
```

## 🔧 高级配置

### 调整UI位置和样式

**位置**: `BossConfig.ui`

```javascript
ui: {
  // 匹配度面板位置
  scorePanel: {
    position: { top: '100px', right: '20px' },  // 👈 调整位置
    autoMinimize: true,     // 5秒后自动缩小
  },

  // 一键打招呼按钮
  quickGreetButton: {
    position: { bottom: '30px', right: '30px' },
    showCondition: 'matched',  // 👈 显示条件：'always' | 'matched' | 'high-score'
  },

  // 主题色
  colors: {
    primary: '#667eea',   // 👈 主色调（紫蓝色）
    success: '#52c41a',   // 👈 成功色（绿色）
  },
}
```

### 调整页面选择器

如果Boss直聘改版导致扩展失效，可以修改选择器：

**位置**: `BossConfig.selectors`

```javascript
selectors: {
  jobList: {
    jobCards: '.job-card-wrapper, .job-card-box',  // 👈 职位卡片选择器
    title: '.job-title, .job-name',  // 👈 职位标题选择器
    // ... 其他选择器
  },
}
```

### 启用调试模式

**位置**: `BossConfig.debug`

```javascript
debug: {
  enabled: true,         // 👈 启用调试模式
  skipDelays: true,      // 👈 跳过所有延迟（快速测试）
  verboseLogging: true,  // 👈 详细日志
  dryRun: true,         // 👈 干跑模式（不实际发送消息）
}
```

**警告**：调试模式会跳过防检测延迟，仅用于开发测试！

## 📊 完整配置项索引

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `defaults.salaryMin` | 期望最低薪资（元/月） | 25000 |
| `defaults.matchThreshold` | 匹配阈值（分） | 60 |
| `defaults.dailyLimit` | 每日打招呼上限 | 30 |

| `behavior.limits.dailyGreet` | 每日硬限制 | 30 |
| `ai.enabled` | 是否启用AI | false |
| `distance.gaode.enabled` | 是否启用距离评估 | false |
| `notification.browser.enabled` | 浏览器通知 | true |

## 💡 配置最佳实践

### 保守配置（推荐新手）

```javascript
defaults: {
  matchThreshold: 60,     // 阈值不要太高
  dailyLimit: 20,         // 每日上限保守一些
}

behavior: {
  delays: {
    beforeClick: { min: 1500, max: 3000 },  // 延迟长一些
  },
  limits: {
    dailyGreet: 20,       // 硬限制20次
    maxContinuous: 3,     // 连续3次就休息
  },
}
```

```javascript
  matchThreshold: 50,     // 降低阈值，看更多职位

```

## ⚠️ 注意事项

### 1. 配置优先级

```
代码配置 (config.js) < 用户UI配置 (popup)
```

- UI配置会覆盖 `defaults` 部分
- 但 `behavior`、`scoring` 等不会被UI覆盖

### 2. 不建议修改的配置

以下配置项**不建议**修改，除非你清楚后果：

- `behavior.delays.typing` - 打字速度太快会被检测
- `behavior.limits.dailyGreet` - 超过50次有封号风险
- `selectors` - 除非页面改版，否则不要改

### 3. API密钥安全

- ⚠️ 不要将包含API密钥的配置文件分享给他人
- ⚠️ 不要提交到公开的Git仓库
- ⚠️ 定期更换API密钥

### 4. 配置备份

修改前建议备份原配置：

```bash
cp config.js config.js.backup
```

## 🆘 配置出错怎么办？

### 恢复默认配置

1. 删除 `config.js`
2. 从Git仓库重新下载原始版本

### 检查配置语法

常见错误：

```javascript
// ❌ 错误：缺少逗号
{
  salaryMin: 25000
  dailyLimit: 30  // 这里缺少逗号！
}

// ✅ 正确
{
  salaryMin: 25000,
  dailyLimit: 30,
}

// ❌ 错误：字符串未加引号
{
  name: 金超宇  // 应该是 '金超宇'
}

// ✅ 正确
{
  name: '金超宇',
}
```

### 查看错误日志

1. 按 F12 打开开发者工具
2. 查看 Console 标签
3. 搜索红色错误信息
